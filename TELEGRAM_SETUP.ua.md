# 📱 Налаштування сповіщень у Telegram

Налаштування Telegram-сповіщень дозволить вам миттєво отримувати повідомлення, коли бекап успішно завершується або якщо виникає помилка.

## Крок 1: Створіть Telegram-бота

1. Відкрийте Telegram та знайдіть бота **[@BotFather](https://t.me/BotFather)**.
2. Надішліть йому команду `/newbot`.
3. Дотримуйтесь інструкцій: вкажіть **Ім'я** (Name) та **Юзернейм** (Username, обов'язково має закінчуватися на `bot`, наприклад, `n8n_backup_alert_bot`).
4. Після створення BotFather надішле вам **HTTP API Токен** (Token). Він виглядає приблизно так:
   `123456789:AABBccDDeeFFggHHiiJJkkLLmmNNoo`
5. **Скопіюйте цей токен**. Він знадобиться для налаштувань у Backup Manager.

## Крок 2: Отримайте ваш Chat ID

Вам потрібно вказати боту, куди саме надсилати повідомлення. Він може надсилати їх безпосередньо вам або в груповий чат.

### Варіант А: Надсилати вам особисто (Direct Message)
1. Знайдіть у Telegram бота **[@userinfobot](https://t.me/userinfobot)** або **[@getmyid_bot](https://t.me/getmyid_bot)**.
2. Натисніть Start. Бот відповість і покаже ваш особистий **Chat ID** (набір цифр, наприклад `123456789`).
3. **Важливо:** Знайдіть свого новоствореного бота (з Кроку 1) і натисніть **Start** (або напишіть йому "привіт"), щоб дозволити йому надсилати вам повідомлення.

### Варіант Б: Надсилати у Групу
1. Додайте вашого новоствореного бота до потрібної Telegram-групи.
2. Додайте бота **[@getmyid_bot](https://t.me/getmyid_bot)** в ту саму групу.
3. Бот миттєво відповість і покаже Group ID. Зазвичай він починається з мінуса (наприклад, `-1001234567890`).
4. Видаліть `@getmyid_bot` з групи після того, як скопіюєте ID.

## Крок 3: Налаштування у Backup Manager

1. Відкрийте **n8n Backup Manager** та перейдіть до **Settings → Notifications**.
2. Поставте галочку **Enable Telegram notifications** (Увімкнути сповіщення Telegram).
3. Вставте **Telegram Bot Token** (з Кроку 1).
4. Вставте **Telegram Chat ID** (з Кроку 2).
5. Натисніть **Send Test Message**. Ви маєте миттєво отримати тестове повідомлення у Telegram!
6. Натисніть **Save Settings** (Зберегти налаштування).
